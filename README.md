# 音乐播放器（MusicPlay）
> 基于PyQt5开发的桌面端音乐播放器，支持丰富的播放控制和列表管理功能

## 简介
本程序是一款跨平台（优先适配Windows）的桌面端音乐播放器，基于PyQt5框架开发，核心功能包括：
- 音频播放核心控制：支持播放/暂停、上一曲/下一曲切换，适配多种音频格式（由配置文件定义）；
- 播放列表管理：支持添加单个音频文件/整个音频文件夹、清空列表、列表排序、拖放调整曲目顺序，右键删除指定曲目；
- 播放模式切换：支持顺序播放、随机播放、单曲循环三种模式；
- 可视化展示：实时显示歌曲进度、音量调节、歌曲元数据（标题/艺术家/专辑）、封面、歌词；
- 系统集成：支持系统托盘操作（播放/暂停、退出），程序运行日志自动记录（按日期生成日志文件）；
- 容错处理：图标加载、文件路径、音频解析等环节均有容错逻辑，提升程序稳定性。

## 安装
### 环境要求
- 操作系统：Windows 10+/macOS 10.15+/Linux（Ubuntu 18.04+，优先适配Windows）；
- 编程语言：Python 3.7+（推荐3.9/3.10，兼容PyQt5最佳）。

### 依赖库安装
程序核心依赖PyQt5框架，需安装指定版本（5.15.9为稳定版）：
```bash
# 安装PyQt5
pip install PyQt5==5.15.9
```

### 额外环境准备
1. 将程序依赖的模块文件（`config_manager.py`、`playlist_manager.py`、`media_core.py`、`play_mode.py`、`meta_data_parser.py`）与`MusicPlay.py`放在同一目录；
2. （可选）配置自定义图标/默认封面：在程序目录下创建`img`文件夹，放入`MusicPlay.ico`（窗口图标）和默认封面图片，或通过配置文件指定路径；
3. 确保系统有读写权限（程序会自动创建`logs`文件夹存储日志）。

## 使用方法
### 1. 基础运行步骤
1. 确认所有依赖文件（上述5个模块）与`MusicPlay.py`在同一目录；
2. 打开终端/命令行，进入程序目录；
3. 运行程序：
   ```bash
   python MusicPlay.py
   ```
4. 程序启动后会自动创建`logs`文件夹并生成当日日志文件，可通过日志排查运行问题。

### 2. 界面操作说明
| 功能区域 | 操作说明 |
|----------|----------|
| 播放列表 | - 「添加文件」：选择单个/多个音频文件加入列表；<br>- 「添加文件夹」：扫描指定文件夹内所有支持的音频文件并加入列表；<br>- 「清空列表」：清空所有曲目并停止播放；<br>- 「排序」：对列表曲目排序（保留当前播放曲目高亮）；<br>- 拖放：直接拖动列表项调整曲目顺序；<br>- 右键菜单：选中曲目右键可删除；<br>- 双击曲目：直接播放选中的曲目。 |
| 播放控制 | - 「上一曲/下一曲」：切换播放曲目；<br>- 「播放/暂停」：控制当前曲目播放状态，点击后按钮文字同步切换。 |
| 音量调节 | 拖动音量滑块调整音量，右侧数字实时显示音量百分比（默认初始音量由配置文件定义）。 |
| 播放模式 | 下拉选择「顺序播放/随机播放/单曲循环」，切换后立即生效。 |
| 歌词显示 | 播放带歌词的音频文件时，歌词会自动加载并显示在右侧歌词面板，当前播放行高亮。 |
| 系统托盘 | 程序最小化后常驻系统托盘，右键菜单可选择「播放/暂停」或「退出」。 |

### 3. 配置文件说明
程序通过`ConfigManager`加载配置（配置文件格式通常为JSON/YAML），核心配置项包括：
```python
# 示例配置项（实际以config_manager.py定义为准）
{
  "window_title": "音乐播放器",
  "window_size": [800, 600],  # 窗口宽、高
  "default_volume": 80,       # 默认音量（0-100）
  "supported_audio_formats": [".mp3", ".wav", ".flac"],  # 支持的音频格式
  "default_cover": "img/default_cover.png",  # 默认封面路径
  "icon_path": "img/MusicPlay.ico"            # 窗口图标路径
}
```

## 示例
### 示例1：添加并播放单个音频文件
#### 操作步骤
1. 运行`python MusicPlay.py`启动程序；
2. 点击播放列表区域的「添加文件」按钮；
3. 选择本地`test.mp3`文件，点击「打开」；
4. 双击列表中的「test.mp3」曲目。

#### 预期结果
- 播放列表中显示`test.mp3`；
- 「播放」按钮变为「暂停」；
- 进度条开始实时更新，当前时间标签从`00:00:00`开始递增；
- 歌曲信息面板显示`test.mp3`的标题、艺术家、专辑（若解析成功），并加载封面；
- 日志文件中记录：`成功添加1个文件到播放列表`、`开始播放：test.mp3`。

### 示例2：切换播放模式并调整音量
#### 操作步骤
1. 按示例1播放`test.mp3`；
2. 在「播放模式」下拉框选择「单曲循环」；
3. 拖动音量滑块至50%位置；
4. 等待曲目播放完成。

#### 预期结果
- 音量标签显示`50%`，音频音量同步降低；
- 曲目播放完成后自动重新开始播放（单曲循环生效）；
- 日志文件中记录：`切换播放模式为：单曲循环`、`调整音量至：50%`。
