# Changelog

## v0.1.0 - 2026-03-17

### 新增功能
- 基于 PyQt5 的桌面音乐播放器
- 播放控制：播放/暂停、上一曲/下一曲
- 播放列表管理：添加文件/文件夹、清空、排序、拖放、右键删除
- 播放模式：顺序播放、随机播放、单曲循环
- 可视化：进度条、音量调节、歌曲信息、封面、歌词显示
- 系统托盘：托盘图标、快捷操作
- 日志记录：按日期生成运行日志
- 元数据解析：支持 FLAC/WAV/MP3/OGG 格式，提取封面和歌词
- 配置管理：支持自定义配置文件
- 打包支持：setup.py 打包脚本
